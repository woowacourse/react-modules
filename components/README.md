# Button Module

## 핵심

- 다양한 형태로 유연하게 사용자가 필요로 하는 정보를 보여준다.

## 기능 구현

- [x] 특정 버튼을 누르면 모달을 띄운다.
- 모달을 닫는다.
  - [x] 모달 내 닫기 버튼을 눌러 닫는다.
  - [x] ESC 키를 눌러 닫는다.
  - [x] 오버레이를 눌러 닫는다.
- [x] 제목을 표시한다.
- [x] 모달 내 내용을 넣는다.
- [x] 모달 내 활용 가능한 버튼 컴포넌트를 만든다.
- 모달에 props를 뚫어 유연한 모달을 만들 수 있다.
  - [ ] 닫기, 확인 등의 동작에 대한 이벤트 핸들러
  - [ ] 상단 닫기 버튼 유무
  - [ ] 확인 버튼 유무
  - [ ] 하단 닫기 버튼 유무
  - [ ] 모달 위치(중앙, 하단)
