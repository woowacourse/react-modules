## 📍 모달

### 기능 요구사항

### 모달 컴포넌트 재사용

- [] 다양한 모달 종류를 대응 가능하게 구현해야 한다.
  - [x] 확인(Alert) 모달
    - [x] 메시지 전달
    - [x] 확인 버튼 제공
  - [] 확인/취소(Confirm)
    - [] 메시지 전달
    - [] 확인 및 취소 버튼 제공
  - [] 입력(Prompt) 모달
    - [] 메시지 전달
    - [] 사용자로부터 입력값을 받을 수 있는 입력 필드
    - [] 확인 및 취소 버튼 제공
- [] 모달 크기 옵션 추가
  - [] small, medium, large 등의 크기 옵션을 prop으로 전달받아 모달 크기 조절
